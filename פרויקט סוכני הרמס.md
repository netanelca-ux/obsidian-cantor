## 🎯 המספרים שצריך לזכור (מהאתר החי mono-assistant-il.netlify.app)
- מותג: **Mono / מונו** · "העוזר האישי לעסק האונליין"
- עלות / לקוח: **~₪85-95 לחודש**
- מנוי: **עסקי ₪279 · פרו ₪559** · **14 ימי ניסיון חינם**
- מודל: **self-serve** ("מתחברים פעם אחת, עובד לתמיד") — *אין דמי הקמה באתר*
- מרווח גולמי: **~75-84%**
- תקרת קיבולת לשני מייסדים ללא גיוס: **~80-120 לקוחות**
- אינטגרציות אמיתיות: **Shopify · PayPlus · Gmail · Google Sheets · WhatsApp · חשבונית**
- יכולות: פותח הזמנות · מפיק חשבוניות · שולח הצעות מחיר · סקירה יומית/שבועית · רץ ברקע

## ✅ החלטות שננעלו
- **מותג:** Mono (הרמס = תשתית הטכנולוגיה מתחת).
- **ערוץ:** מספר WhatsApp רגיל (חיבור לא-רשמי), מספר מבודד לכל לקוח. **לא** API רשמי של Meta.
- **מספרים:** eSIM מ-019, הקצאה מיידית ומרחוק.
- **AI:** פלטפורמת Codex (מנוי פר-לקוח).
